Welcome to an introductory Mobile Forensics Lab, created by Maximillian Carney and Noah Cacio of Champlain College

The hope of our project is to create an introductory lab for any aspiring investigator to understand the basics of mobile forensics and how it ties into law enforcement
This lab follows a murder case in Northern VT. Images of the phones and crime scene photos have been provided. It is up to you, the investigator, to put the case together and find out what happened
Located within this repository is access to the hands-on lab, a presentation for the background information, as well as image files needed to complete the lab (Images in google drive due to GitHub file size restrictions)
Additionally there is a locked file that contains the answer key to the lab in case anyone struggles to find any information. The lab should have extensive instructions on how to get the proper access to free to use tools in case you do not have a proper Axiom key

Feel free to download the images, the lab, and the read me, and you should be all set to start!

After completing the lab, also feel free to check out the legal-documents folder located in the GitHub. There we have written attached legal documents that would be created by a PD with the information from this investigation. This includes an arrest warrant, search warrant, subpoena, media release, and a chain of custody. We worked closely with law enforcement to create these documents to ensure we could produce the most realistic product as possible. We hope this can clue people into the legal side of a digital forensics investigaiton.
